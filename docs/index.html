kubetray

Local K8s, served on a tray.

Install

# Homebrew
brew install ugurozkn/tap/kubetray

# npm
npx kubetray start

# From source
git clone https://github.com/ugurozkn/kubetray.git
cd kubetray && make install

Commands

kubetray start                       # Start cluster
kubetray start --cpus 4 --memory 8G  # Custom resources
kubetray stop                        # Stop cluster
kubetray clean                       # Delete cluster
kubetray version                     # Version info

MCP (AI Integration)

// ~/.claude/claude_desktop_config.json
{
